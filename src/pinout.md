# Connectors and Schematics:
## HDMI Out
### HDMI Connector to J10:

TMDS0_p -> A1_p -> Pin 20 \
TMDS0_n -> A1_n -> Pin 22 

TMDS1_p -> A2_p -> Pin 26 \
TMDS1_n -> A2_n -> Pin 28 

TMDS2_p -> A3_p -> Pin 32 \
TMDS2_n -> A3_n -> Pin 34 

CLK_p   -> B1_p -> Pin 21 \
CLK_n   -> B1_n -> Pin 23
